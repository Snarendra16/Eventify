  # Do following steps to properly run this project
***
- **Step 1**: Install all dependencies ($npm install)
- **Step 2**: Go to virtual-event-platform directory ($ cd virtual-event-platform/)
- **Step 3**: Run frontend. ($npm start)
- **Step 4**: Go to backend directory ($ cd backend/)
- **Step 5**: Run backend ($node server.js)
- **Step 6**: Run the application. ($npm start)
